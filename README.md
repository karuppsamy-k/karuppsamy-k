## About Me

I'm an **Android SDK Engineer** with 3+ years of production experience building
**camera SDKs** and **React Native native modules** for enterprise clients,
including **Honeywell** and **Rockwell Automation**.

- 🎥 I build production-grade **camera & document-scanning SDKs** using CameraX and Camera2, packaged as modular AARs for third-party integration
- 🌉 I design **JSI-compatible native bridges** connecting Android SDKs to React Native
- 🏗️ I care about **clean, versioned SDK architecture** — public API surfaces, integration contracts, and zero-escalation self-integration for client teams
- 🧪 I write **JUnit test suites** and use the Android Studio Profiler to hunt down memory leaks, ANRs, and crash clusters
- 📦 I own SDKs end-to-end: API design → implementation → QA → **Google Play Console** release lifecycle

## Current Focus

- Deepening public-facing documentation and sample repositories for the SDK architecture patterns I use in production (see pinned repos)
- Exploring React Native's New Architecture (TurboModules + JSI) beyond the proof-of-concept stage
- Formalizing internal Clean Architecture / Hilt DI patterns into reusable, shareable templates
