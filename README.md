**camera SDKs** and **React Native native modules** for enterprise clients,
including **Honeywell** and **Rockwell Automation**.

- 🎥 I build production-grade **camera & document-scanning SDKs** using CameraX and Camera2, packaged as modular AARs for third-party integration
